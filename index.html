hexo学习心得

github仓库创建

和普通仓库创建一样但仓库名必须为本人github的id。如我的id是”shaoalong”，
所以仓库名为”shaoalong.github.io”。

新建秘钥

1 . git bash输入

其中”youremail@example.com”是本人的邮箱
可以在用户主目录里找到.ssh目录，里面有id_rsa和id_rsa.pub两个文件。这两个就是SSH Key的秘钥对，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。
2 . 登陆GitHub，打开“Account settings”，“SSH Keys”页面，然后点“Add SSH Key”，填上任意Title，在Key文本框里粘贴id_rsa.pub文件的内容。

本地建文件夹 (hexo 给予node)

1 . 新建一个文件夹用来放置hexo文件 如我的文件夹叫hexo
2 . 安装hexo： npm install -g hexo
3 . 初始化hexo： hexo init
之后文件夹会有以下文件:

4 . 生成静态页面： hexo generate(hexo g:提交到github，没有这个步骤就不会提交到github上，只会在本地localhost：4000服务器上有效)
6 . hexo 部署： hexo deploy(hexo d)
7 . 打开localhost：4000服务器 ：hexo server(hexo s)

hexo关联github

找到站点config.yml，在文件的最下面增加形如内容

deploy:
type: git
repository: git@github.com:shaoalong/shaoalong.github.io.git
branch: master

其中：git@github.com:shaoalong/shaoalong.github.io.git 是本人的git仓库
然后执行 npm install hexo-deployer-git --save即可。

主题设置

把主题包解压到theme目录下可随意改名字如yilia，在站点config.yml找到theme并设置为该主题文件的名字yilia即可
